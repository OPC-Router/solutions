Mit dieser Solution können Sie über einen Workflow in Teams eine einfache Textbenachrichtigung als dynamische Karte an Microsoft Teams versenden.  
Das Senden eines dynamischen Textes kann in der Verbindung konfiguriert werden. Auch das Ändern der Karte ist durch das Anpassen des JSON Schreiben Transferobjektes einfach möglich.

##### Voraussetzungen für das Verwenden dieser Solution

* Workflows müssen in Ihrem Teams verfügbar sein.

##### Anleitung zum Einrichten der Benachrichtigungen mit Workflows in Microsoft Teams

1. Klicken Sie bei einem Chat oder einem Kanal auf das ... Menü und wählen Sie "Workflows" oder öffnen Sie die Workflows App.
2. Klicken Sie auf "Alle Vorlagen".
3. Suchen Sie nach "Webhook" und wählen Sie entweder "Webhookbenachrichtigungen an einen Kanal senden" oder "Webhookbenachrichtigungen an einen Chat senden".
4. Konfigurieren Sie den Kanal oder Chat, an den die Benachrichtigung gesendet werden soll.
5. Klicken Sie auf "Speichern".
6. Kopieren Sie den Webhooklink mit dem Button "Webhooklink kopieren".

##### Tipp

Zum Entwerfen eines anderen Kartenlayouts können Sie den [Designer für Adaptive Karten](https://adaptivecards.microsoft.com/designer) von Microsoft verwenden.
