With this Solution, you can send a simple text notification as a dynamic card to Microsoft Teams via a Workflow in Teams.  
Sending a dynamic text can be configured in the connection. Changing the card is also easy by customizing the JSON Write Transfer Object.

##### Requirements for using this solution

* Workflows must be available in your Teams.

##### Instructions for setting up notifications with Workflows in Microsoft Teams

1. Click on the ... menu in a chat or channel and select “Workflows,” or open the Workflows app.
2. Click on “All Templates.”
3. Search for ‘Webhook’ and select either “Send webhook notifications to a channel” or “Send webhook notifications to a chat.”
4. Configure the channel or chat to which the notification should be sent.
5. Click “Save.”
6. Copy the webhook link using the “Copy webhook link” button.

##### Tip

To design a different card layout, you can use Microsoft's [Adaptive Cards Designer](https://adaptivecards.microsoft.com/designer).
