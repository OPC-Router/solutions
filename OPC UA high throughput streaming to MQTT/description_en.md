This Solution contains an **OPC UA client plug-in** optimized for the **Multi Datachange Trigger**, which is optimized for **high data throughput**.

##### Optimizations made

* OPC UA Client
  * Asynchronous browsing for the Multi Datachange Trigger is used.
* Multi Datachange Trigger
  * Updating of items has been disabled.

The included connection transfers data from the Multi Datachange Trigger to an MQTT broker.
