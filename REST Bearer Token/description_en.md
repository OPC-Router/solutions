This Solution contains a simple structure for requesting and using a Bearer Token for authentication with the REST Transfer Object.

The Solution includes two connections. One connection is used to request the Bearer Token and store it in a variable. The other connection is used for Bearer Token authentication.

##### Note

* The GET method is used to request the token in the structure. If a **request body** is required for the request, another method that supports request bodies, such as **POST**, must be used.
