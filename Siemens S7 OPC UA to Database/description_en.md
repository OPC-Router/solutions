This Solution contains an OPC UA client plug-in optimized for connecting to a Siemens S7 PLC. A connection enables data transfer from the Siemens S7 controller to an MSSQL database.

The following measures were implemented in the plug-in for connecting to an S7 controller:

* The OPC UA client uses a client certificate
* The addressing method was changed to Node ID

##### Note

* The data points and table columns must be configured manually after importing the solution.
