Mit dieser Solution können Sie über einen Webhook in Slack Nachrichten senden.
Das Senden eines dynamischen Textes kann in der Verbindung konfiguriert werden. Auch ein Ändern des Designs von der Nachricht ist durch das Anpassen des JSON Dokuments im JSON Schreiben Transferobjekt möglich.

##### Voraussetzungen für das Verwenden dieser Solution

* Es muss eine Slack App geben, die eine Webhook-Adresse hat
* Diese Adresse muss bekannt sein

##### Erstellen des Webhooks

Eine [Anleitung zum Senden von Nachrichten über Webhooks](https://docs.slack.dev/messaging/sending-messages-using-incoming-webhooks/) befindet sich in der Dokumentation von Slack. In der Anleitung wird erklärt, wie der zum Senden der Nachrichten benötigte Webhook erstellt wird.
