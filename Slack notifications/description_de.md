Mit dieser Solution können Sie über einen Incoming Webhook Benachrichtigungen an Slack senden.
Das Senden eines dynamischen Textes kann in der Verbindung konfiguriert werden. Auch ein Ändern des Designs von der Nachricht ist durch das Anpassen des JSON Dokuments im JSON Schreiben Transferobjekt möglich.

##### Voraussetzungen für das Verwenden dieser Solution

* Es muss eine Slack App mit einem Incoming Webhook geben

##### Erstellen des Webhooks

Eine [Anleitung zum Senden von Nachrichten über Webhooks](https://docs.slack.dev/messaging/sending-messages-using-incoming-webhooks/) befindet sich in der Dokumentation von Slack. In der Anleitung wird erklärt, wie der zum Senden der Nachrichten benötigte Incoming Webhook erstellt wird.
