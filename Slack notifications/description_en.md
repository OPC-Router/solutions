With this Solution, you can send notifications to Slack via an incoming webhook.
Sending dynamic text can be configured in the connection settings. You can also customize the message's design by modifying the JSON document in the JSON write transfer object.

##### Prerequisites for using this Solution

* A Slack app with an incoming webhook must be available

##### Creating the webhook

[Instructions for sending messages using incoming webhooks](https://docs.slack.dev/messaging/sending-messages-using-incoming-webhooks/) can be found in the Slack documentation. The guide explains how to create the incoming webhook required for sending messages.
