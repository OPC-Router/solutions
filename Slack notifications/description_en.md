With this Solution, you can send messages via a webhook in Slack.
Sending dynamic text can be configured in the connection. You can also change the design of the message by customizing the JSON document in the JSON Write Transfer Object.

##### Requirements for using this Solution

* There must be a Slack app that has a webhook address
* This address must be known

##### Creating the webhook

[Instructions for sending messages using webhooks](https://docs.slack.dev/messaging/sending-messages-using-incoming-webhooks/) can be found in the Slack documentation. The instructions explain how to create the webhook required for sending messages.
